# Vim Pro
## Benefit
- Cool
- Exist on all Unix system
- Productivity

## Disadvantage
- Addictive

## Tools
- neovim
- vim

## Commands
:q - exit
:w <file_name>- save
:wq - save & exit
:q! - force exit
:wq! - force save & exit

## Mode
- normal mode
- visual mode
- insert mode

## How to move in vim
- j - Up
- k - Down
- h - Left
- l - Right

## How to change into insert mode
- i - insert
- I - prepend
- a - append
- r - replace 
- R - replace all

## How to remove 
- x - remove
- <number>x - remove x charactor
- dd - remove line
- dw - remove word

## Back to normal mode 
- esc

## how to change into visual mode
- v - visual (drag)
- d - remove
- y - copy 
- p - paste
- U - upper case
- u - lower case
